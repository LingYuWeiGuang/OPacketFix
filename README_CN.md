# OPacketFix
一个基于Mixin的PacketFixer，支持Minecraft-1.8.9 Forge，用于中国服务器花雨庭。

必须同时安装ViaForge模组.

## 开发指南
1. 在 Windows 系统中使用 Intellij IDEA 和 Gradle 。
2. 在 cmd 中输入：gradlew build
3. 打开 IDEA 并等待它自动同步。
